# پلن من — ۴۰ روز 🚀

برنامه شخصی PWA برای مدیریت ۴۰ روز کنکور + باشگاه + عادت‌های روزانه + موزیک.

## ساختار پروژه

```
myplan/
├── index.html          ← فایل اصلی (بازش کن)
├── manifest.json        ← تنظیمات PWA
├── sw.js                 ← سرویس‌ورکر (کارکرد آفلاین)
├── css/
│   └── style.css        ← تمام استایل‌ها (۲۰+ رنگ، افکت سه‌بعدی)
├── js/
│   ├── data.js          ← نقل‌قول‌ها، برنامه باشگاه، تنظیمات روزها
│   ├── particles.js     ← انیمیشن پارتیکل پس‌زمینه + کانفتی
│   ├── audio.js         ← موزیک پلیر (هر روز یه آهنگ جدا)
│   ├── habits.js        ← عادت‌تراکر
│   ├── plan.js          ← برنامه شخصی (تودو)
│   └── app.js           ← منطق اصلی، تقویم، صفحات
└── assets/
    ├── bg.png           ← تصویر پس‌زمینه
    ├── icon-192.png     ← آیکون PWA
    └── icon-512.png     ← آیکون PWA
```

## نصب روی آیفون (مثل یک اپ)

1. تمام پوشه `myplan` رو آپلود کن روی یه هاست ساده (مثلاً GitHub Pages،
   Netlify، یا حتی یه سرور لوکال).
2. لینک `index.html` رو با **Safari** باز کن (نه کروم).
3. دکمه Share (مربع با فلش رو به بالا) رو بزن.
4. **Add to Home Screen** رو انتخاب کن.
5. تمام! حالا یه اپلیکیشن کامل روی صفحه اصلی داری.

## ویژگی‌ها

- 🎵 **موزیک پلیر واقعی**: برای هر روز می‌تونی یه فایل صوتی جدا انتخاب کنی
  و توی همون روز پخش بشه (با کنترل play/pause، نوار پیشرفت و ولوم).
- 📅 تقویم کامل ۴۰ روزه با جزئیات هر روز (برنامه ساعتی، باشگاه، چک‌لیست).
- 💧 ردیاب آب روزانه با انیمیشن لیوان‌ها.
- 📊 حلقه پیشرفت کلی + پیشرفت هر فاز.
- 💬 نقل‌قول‌های روزانه با چرخش خودکار.
- ⚡ عادت‌تراکر با محاسبه streak (روزهای پشت‌سرهم).
- 📋 بخش "برنامه من" کاملاً قابل شخصی‌سازی — هر بخش و آیتمی که بخوای اضافه کن.
- ⚙️ بکاپ‌گیری و بازیابی کامل داده‌ها (JSON).
- 🌌 پس‌زمینه پارتیکل تعاملی + گلوهای رنگی + بافت نویز.
- 📱 کاملاً ریسپانسیو و قابل نصب به‌صورت PWA (آفلاین هم کار می‌کنه).

## نکته مهم درباره موزیک

فایل‌های صوتی به‌صورت Base64 داخل `localStorage` مرورگر ذخیره می‌شن.
حجم `localStorage` معمولاً حدود ۵-۱۰ مگابایت محدود است، پس بهتره
فایل‌های صوتی کوتاه (کمتر از ۱-۲ مگابایت) انتخاب کنی. اگه حافظه پر شد،
یه پیام هشدار نمایش داده می‌شه.

## شخصی‌سازی

- نقل‌قول‌ها، برنامه باشگاه، نکته‌های روزانه و... همه توی `js/data.js` هستن.
- رنگ‌ها و افکت‌های ظاهری توی `css/style.css` (متغیرهای `:root`).
- می‌تونی هر چیزی رو از داخل اپ هم (دکمه‌های + اضافه‌کن) شخصی‌سازی کنی،
  بدون نیاز به تغییر کد.
